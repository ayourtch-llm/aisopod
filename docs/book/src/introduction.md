# Introduction

Welcome to the Aisopod documentation.

Aisopod is an AI-native messaging gateway that bridges the gap between conversational AI and traditional communication channels. Built with Rust, Aisopod provides a robust, type-safe foundation for building AI-driven messaging applications.

## What is Aisopod?

Aisopod enables developers to:

- Connect AI agents to multiple messaging channels (Slack, Discord, Matrix, etc.)
- Build intelligent conversational workflows with configurable agents
- Deploy secure, scalable messaging infrastructure
- Integrate AI capabilities into existing communication systems
